## 柑橘品種情報

### 1. まどんな
- **正式名称**: 愛媛果試第28号
- **説明**: 愛媛県生まれの県オリジナル品種。ゼリーのようなとろけるような食感と、甘い果汁と豊かな香りが特徴。収量が限られる高級柑橘。
- **モーダル画像**: 3枚

### 2. はれひめ
- **説明**: 果肉が柔らかく、つぶつぶがプリッとした食感で、オレンジの爽やかな風味が特徴。同じ木でもみかんの様な形とオレンジの様な形のものが混在。
- **モーダル画像**: 2枚

### 3. 媛小春・甘平
- **説明**: 愛媛県限定品種。媛小春は酸味が少なく爽快な風味。甘平は外皮が薄く栽培管理が難しい品種。
- **モーダル画像**: 2枚

### 4. その他の品種
- **説明**: レモン、温州みかんなど、古来から親しまれている柑橘。
- **モーダル画像**: 2枚

## 加工品情報

### ジュース
インライン搾汁製法で、柑橘本来の味を雑味なくぎゅっと絞った果汁100%ストレートジュース。酸味料などの添加物不使用。

### はちみつ
2019年より養蜂開始。みかん・伊予柑・八朔の花々から集めた非加熱はちみつ。

| `--olive-dark` | #6B7B3D | フッター背景、文字色 |

## 機能詳細

### ヒーロースライダー
- 3枚の画像を5秒間隔で自動切り替え
- 左右矢印とドットでの手動切り替え可能

### モーダル（ポップアップ）
- 各商品カードをクリックで詳細表示
- モーダル内に画像スライダー（2〜3枚）
- ESCキーまたは背景クリックで閉じる

### レスポンシブ対応
- モバイルではハンバーガーメニュー
- 画像グリッドは画面幅に応じて2列→1列に変化

### ご購入ページ特有の設定
- `min-height: 100vh` + flexbox
- コンテンツ上部、フッターが画面下部に配置

## 画像について

### 注意点
- HEICファイルは**変換が必要**（ファイル名変更だけでは不可）
- Macのプレビューで「ファイル」→「書き出す」→「JPEG」で変換
- 推奨サイズ: 1600〜2000px幅、200〜500KB

### ファビコン
- サイズ: 32×32px または 64×64px
- 形式: PNG（透過可）
- ロゴを正方形にトリミングして使用

## 次のステップ: GitHubとVercelへのデプロイ

### 1. GitHubリポジトリ作成
```bash
cd ~/Desktop/setouchi\ LIETO
git init
git add .
git commit -m "Initial commit"
```

### 2. GitHubでリポジトリ作成
- https://github.com/new でリポジトリ作成
- リポジトリ名: `setouchi-lieto`（推奨）

### 3. GitHubにプッシュ
```bash
git remote add origin https://github.com/[ユーザー名]/setouchi-lieto.git
git branch -M main
git push -u origin main
```

### 4. Vercelでデプロイ
1. https://vercel.com にログイン
2. 「New Project」をクリック
3. GitHubリポジトリを選択
4. デプロイ設定はデフォルトでOK（静的HTML）
5. 「Deploy」をクリック

### 5. カスタムドメイン設定
- Vercelダッシュボード → Settings → Domains
- `setouchi-lieto.com` を追加
- DNSレコードをVercelの指示通りに設定
